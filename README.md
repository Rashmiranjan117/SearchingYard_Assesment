## Want to run this project Locally?

- Clone this project
 
 ```bash
 git clone https://github.com/Rashmiranjan117/SearchingYard_Assesment.git
 ```
 
 - Install all Packages (Both Frontend and Backend)
  ```
  cd frontend
  npm install
  npm run dev
  ```
  
  ```
  cd backend
  npm install
  npm run server
  ```
